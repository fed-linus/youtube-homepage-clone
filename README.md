**YouTube Page Clone**

A simple YouTube homepage clone built with HTML and CSS. This project replicates the layout and structure of YouTube’s homepage, including a header, sidebar navigation, and a grid of video previews with thumbnails, titles, authors, and view counts.

You can visit the page live at [YouTube Homepage Clone](https://linus9426.github.io/youtube-homepage-clone/)

---

**Features**

* Fully responsive header with search bar, voice search, upload icon, notifications, and profile picture.

* Sidebar navigation mimicking YouTube’s menu (Home, Explore, Subscriptions, Originals, Music, Library).

* Video grid layout with video thumbnails, titles, authors, and view statistics.

* Clean separation of CSS files for general styles, header, sidebar, and video content.

---

**Technologies Used**

* HTML5

* CSS3

---

**Notes**

* This project is for learning and demonstration purposes only.

* All video content and channel images are static placeholders.

* No JavaScript functionality is implemented (purely frontend layout).
